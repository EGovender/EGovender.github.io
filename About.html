US Visa Applications

About the Visualization

Hybrid structure
This interactive story of US Visa Applications follows a Martini Glass narrative style visualization.

We first begin by exploring the data from a high level by examining the origin and destination of applicants. In particular the focus in on the Top 20 Countries with the highest number of applications. We see popular destinations that applicants will be based at should their application be successful.

A general pattern of migration is observed, with people heading either to the West Coast, East Coast or Texas. Questions such as what are the deciding factors for these patterns arise.
The user has the ability to select one of the Top 20 countries and the data changes as a result of this selection. The user is however constrained by the author-driven type of information presented.

The bar chart in the proceeding slide is purely author driven. It presents a chart showing the percentage per State of the Total Applications received. It clearly highlights using annotation that California is by far the most popular destination State in applications.

Finally the story ends with the bubble chart where the user is able to explore the data more interactively. This is achieved in two ways:
• A drop-down selection of States is provided so the user can filter the information.
• A drill down capability to take a deeper look at the statistics for a State and Job Category.
The user is able to decide the path they would like to follow through this interaction. Thereby completing the Martini Glass structure for this narrative.

Use of Scenes
This visualization consists of 3 scenes. Scenes are Numbered 1 to 3 and navigation is via buttons at the top of every slide.

Annotations
The map and bar charts are clearly annotated drawing attention to cities and states of interest respectively.

Parameters
Scene 1:
• A selection of filters for Countries is provided so the user can filter the information.

Scene 3:
• A drop-down selection of States is provided so the user can filter the information.
• A drill down capability to take a deeper look at the statistics for a State and Job Category.

Triggers
Scene 1:
• A change to the selection of filters for Countries changes the scene depiction and annotations.

Scene 3:
• A change in drop-down selection of States changes the information displayed.
• A drill down capability changes the level of granularity that the information is displayed at.
